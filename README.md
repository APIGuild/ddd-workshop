# ddd-workshop [![Build Status](https://travis-ci.org/APIGuild/ddd-workshop.svg?branch=master)](https://travis-ci.org/APIGuild/ddd-workshop)

### Install Plugins
+ Lombok
+ MapStruct

### Start continuous build and run
```
./gradlew build --continuous

./gradle bootRun
```
